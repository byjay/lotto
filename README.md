# 로또봉봉 (Lotto BongBong) 🎱

**로또봉봉**은 동행복권의 실제 당첨 데이터를 기반으로 통계적 분석과 AI 알고리즘을 통해 로또 번호를 추천해주는 웹 애플리케이션입니다.

## ✨ 주요 기능

*   **실시간 데이터 연동**: 동행복권 API를 통해 최신 당첨 번호를 자동으로 가져옵니다.
*   **다양한 분석 모드**:
    *   **빈도 분석**: 역대 가장 많이 나온 번호를 분석합니다.
    *   **트렌드 분석**: 최근 당첨 번호의 합계 추세를 그래프로 보여줍니다.
    *   **홀짝/고저 분석**: 번호의 홀/짝 비율과 고/저 비율을 시각화합니다.
*   **스마트 AI 추첨**:
    *   **최적 조합 (Deterministic)**: 통계적으로 가장 유력한 번호 조합을 제시합니다.
    *   **가중치 추천 (Probabilistic)**: 빈도 기반 가중치를 적용하여 랜덤성을 가미한 추천을 제공합니다.
*   **나의 번호 관리**: 생성한 번호를 저장하고, 실제 당첨 결과와 비교해볼 수 있습니다. (LocalStorage 사용)

## 🛠 기술 스택

*   **Frontend**: HTML5, CSS3 (Tailwind CSS), JavaScript (Vanilla)
*   **Visualization**: Chart.js
*   **Font**: Google Fonts (Jua)
